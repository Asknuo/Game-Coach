# 换血时机

## 换血窗口
1. **对方补刀时** — 攻击动画中无法还手（最优窗口）
2. **对方关键技能CD中** — 尤其是位移/控制技能
3. **你有技能强化普攻** — 贾克斯 W、猴子 Q、剑姬 E
4. **对方没蓝** — 法师没蓝 = 步兵
5. **对方刚交完 TP** — 状态一般不会满

## 不该换血的时候
1. 对面有大量小兵（前期兵伤害 = 半个人）
2. 对面打野在小地图上消失了
3. 你闪现 CD 中（没退路）
4. 对面有盾/屏障/治疗术

## 换血公式
```
换血结果 = 你造成的伤害 - 对方造成的伤害 - 对方小兵伤害
```
小兵伤害：前期每个近战兵 12/s、远程兵 24/s、炮车 40/s。
换血 > 0 = 赚，持续换血即滚雪球。
