# 兵线管理

## 四种兵线操作
| 操作 | 做法 | 何时用 |
|------|------|--------|
| **快推** | 用技能清兵，不控蓝 | 要回城 / 要游走 / 对方回城了 |
| **慢推** | 只补最后一刀 | 发育期 / 等打野来 gank |
| **冻结** | 只补最后一刀，用身体挡住对方兵 | 对线劣势但不想被抓 / 领先时控线折磨对方 |
| **重置** | 把一波兵全部推进塔 | 对方刚被杀 / 强迫对方亏一波兵 |

## 什么时候该推线
- 你杀了对方 → 快推进塔，让对方亏一波
- 对方刚回城 → 快推进塔然后回城同步
- 打野在附近想入侵对方野区 → 推线让你能支援
- 要打小龙/峡谷先锋 → 提前 15 秒把线推进去

## 什么时候该控线
- 对面打野没出现过（防 gank）
- 你领先对面但杀不了他（让他吃不到刀 > 杀他）
- 己方打野要去 gank 隔壁路（不需要你帮忙）
- 对面没闪/没净化（等打野来就能杀）

## 什么时候该冻线
- 对面英雄前期比你强但你发育起来更强（狗头、卡萨丁）
- 对面刚交完闪现
- 己方打野在地图上 ping 说要来
