# 关键眼位

## 对线期必插眼位（按优先级）
1. **河道草** — 最基本，防 gank 第一道防线（1分30秒插）
2. **三角草** — 蓝色方上路/红色方下路更重要
3. **对方蓝buff/红buff入口** — 追踪打野位置（2分50秒左右插）
4. **中路两侧河道** — 辅助/打野路过时帮忙补

## 中期转线眼位
1. 敌方野区入口（F4/六鸟入口）
2. 龙坑上方草丛
3. 中路两侧河道草

## 视野常识
- 控制守卫（真眼）插在己方野区入口比插在河道更持久
- 扫描饰品在中后期比远见宝珠更重要（排视野 > 放视野）
- 辅助的眼石任务完成后，假眼插在关键路口而非草丛更有效
