# 回城时机详解

## 必须立即回城的情况
1. **血量 < 15%**：任何英雄一套技能+点燃都能杀
2. **无视野+中路消失**：大概率有人在蹲你
3. **对方打野带着红蓝buff正在靠近**（小地图可见）
4. **刚躲过 gank 交了闪现**：第二次你一定死

## 可以贪一波的情况
1. **对方刺客刚交了全套技能没杀掉你**
2. **己方打野打信号说正在路上了**
3. **对方也在残血补刀，谁先回谁亏线**
4. **你有药水且对方没有点燃**

## 完美回城的操作
1. 等一波炮车线到来
2. 快速清掉这波线（或者用技能硬推）
3. 自己的兵线推进对方塔下
4. 趁对方清炮车的时候 B 回城
5. 走回线上最多亏 1-2 个近战兵
