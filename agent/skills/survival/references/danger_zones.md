# 危险区域判断

## 绝对危险区（无视野 = 死亡）
- 河道草丛（both sides）
- 三角草（己方塔被推掉后）
- 蓝buff / 红buff 入口草丛（对面打野最爱的蹲点）
- 一塔和二塔之间的草丛

## 相对危险区（需要信息才能判断）
- 己方塔前 500 范围内的区域（取决于打野位置）
- 河道中心（取决于中路是否有线权）
- 敌方三狼/六鸟附近（入侵时）

## 安全区判断公式
```
安全度 = 己方视野覆盖 + 己方防御塔存在 + 己方队友在线 - 敌方失踪人数 × 2
```
安全度 < 0 时，立即撤退。

## 打野追踪速查
- 常规开野：蓝方红开 → 3分钟到上路；红方蓝开 → 3分钟到下路
- 速三 gank：2分30秒就可能出现在线上
- 没出现在地图上超过 30 秒 → 大概率在蹲你
