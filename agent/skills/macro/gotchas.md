# Macro 坑点清单

## ARAM 陷阱
- **不要建议"集合打团"**——低分段这样只会变成 5 个人中路站街，分不到经验和金币
- 如果队友已经在中路大乱斗，提醒他们去边线收兵——边路线是免费的经验和金币
- 中期的最佳节奏是 1-3-1 或 4-1，不是 5 个中路怼 5 个

## 分带迷思
- 分带的黄金法则：**等对面出现在地图上再推进**，不是"我有 TP 就可以无脑带"
- 对面少 2 人 → 抱团推塔 > 分带（5v3 攻塔必胜）
- 对面全活着 → 分带牵制 > 抱团
- 你的 TP 还有 180 秒 CD → 别去下路分带，对面开大龙你回不来

## 目标优先级
- 第二条峡谷先锋换第二条龙是**亏的**——先锋能撞 3-5 层塔皮 + 可能破一塔
- 中一塔比任何一条龙都重要——中一塔被破 = 你方地图少了一半视野
- 大龙对面被团灭 3 人以上时，**推高地 > 打大龙**。推高地是确定的收益，大龙有可能被抢

## 大龙决策
- 永远不要建议"5人打大龙"——正确的做法是"逼团不逼龙"
- 大龙区的团战死亡不是 1+1=2，死 2 个人就可能被一波
- 打完大龙立刻回城补状态——不回城贪推塔 = 被复活回来的对方收掉 buff
