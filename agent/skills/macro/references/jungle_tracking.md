# 打野追踪

## 初始位置判断（1:30）
- 敌方下路/上路先上线 → 没有帮打野开野
- 反之 → 打野从另一侧开始

## 打野时间表
| 时间 | 打野可能在哪 |
|------|------------|
| 2:30 | 速三 gank  |
| 3:15 | 常规全清后第一次 gank |
| 4:00 | 第一轮野区清完，可能在争河道蟹 |
| 7:30 | 红/蓝 buff 刷新，打野大概率在 buff 区 |
| 8:00 | 峡谷先锋刷新，可能在上半区 |

## 对方打野位置推断
- 对方打野出现在小地图 → 记下时间和位置
- 30 秒没出现 → 80% 概率在另一侧
- 对方辅助消失 → 可能是和打野一起行动
- 对方打野助攻一条龙后 → 30 秒内会回到自家野区

## 入侵时机
- **最佳入侵时机**：对方打野刚出现在小地图另一侧
- **次佳入侵时机**：你刚杀了对方打野
- **不要入侵**：己方对线路没有线权时
