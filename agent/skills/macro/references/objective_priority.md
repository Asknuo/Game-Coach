# 目标优先级

## 目标价值排名（通用）
```
1. 远古龙（斩杀效果 = 必胜团）
2. 大龙（180秒 buff + 300全队金币）
3. 龙魂点（第3条龙 = 全队永久buff）
4. 中一塔（控制一半地图视野）
5. 第二/三条龙
6. 峡谷先锋（塔皮=金币、推塔能力）
7. 第一条龙
8. 边路一塔
```

## 什么时候换目标
| 交换 | 赚还是亏 |
|------|----------|
| 峡谷先锋换第一条龙 | 赚（先锋能破3层塔皮=480g） |
| 第二条峡谷先锋换第二条龙 | 赚（先锋能撞高地 vs 一条龙buff） |
| 中一塔换一条龙 | 赚（中塔视野控制永久 vs 一个buff） |
| 大龙换远古龙 | 亏（远古龙 >>> 大龙） |
| 下路一塔换上路一塔 | 看情况（下塔近龙 = 龙区控制更强） |

## 目标决策树
1. 远古龙活着？→ 远古优先
2. 大龙活着？→ 做视野，别开
3. 龙魂点要刷了？→ 提前60秒集合
4. 峡谷先锋还有？→ 13分钟内必须用掉
5. 对面少人？→ 推塔 > 打龙
