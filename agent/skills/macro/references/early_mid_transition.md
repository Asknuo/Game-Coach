# 对线期→中期转换（14 分钟）

## 对线期结束的 3 个信号
1. 任一团队拿到了 5 层塔皮或推掉了一塔
2. 游戏时间 14 分钟
3. 任一团队 3 人以上集结在某条路

## 各位置转换职责

### Top（上单）
- 一塔推掉 → 去对面野区做视野 → 回城 → 走下路带线
- 一塔还在 → 继续对线，但注意地图下半区（准备 TP）
- 自己有 TP 队友没 → 你去带远离下一个目标的线

### Mid（中单）
- 此时是你 power 最足的时期（先到了 11 级 + 2 件套）
- 清完中线 → 立刻和打野一起游走
- 中一塔是生命线——保护中一塔 > 去边路帮队友

### ADC + Support
- 推完下路塔 → 立刻转中路（给中路去边线带的空间）
- 辅助在转线过程中要一路排视野过去

### Jungle（打野）
- 峡谷先锋 14 分钟前必须用——除非在打更重要的东西
- 重点控制龙坑视野
- 时刻关注小地图 ping 出对方打野位置

## 经典错误
- 5 人站在中路大乱斗（ARAM）→ 只会分不到经验和经济
- ADC 继续在下路 farm → 会被 3 人包夹，中路也会丢塔
- 打完团赢了 → 不推塔而是集体回城买装备
