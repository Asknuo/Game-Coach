# 各位置团战站位

## Top（上单）
- 坦克：站在团队正前方 300-500 单位处，吸收伤害和控制
- 战士：站在侧面等待进场时机，不要正面上——会被风筝
- 分带型：团打起来时立刻传送（或提前确认团战会打就TP）

## Jungle（打野）
- 坦克打野：同上单坦克
- 刺客打野：站在侧后方等对方交了关键技能再进场
- **打野最重要的团战职责**：惩戒对方关键目标（龙/大龙）或者在团输时抢龙止损

## Mid（中单）
- 法师：和 ADC 保持 300-400 单位距离，站在侧后方
- 刺客：和上单战士相同的侧翼站位
- 辅助型中单（加里奥/卡牌）：站得靠前，随时准备进场保护

## ADC（Bot）
- 永远站在最后面——你的输出距离 = 你的存活范围
- 保持你最大攻击距离的边缘
- 输出最安全的目标，别为了打 carry 把自己暴露在危险中

## Support（辅助）
- 软辅：站在 ADC 和敌人之间，随时准备给盾+加速+解控
- 硬辅：站在前排和后排之间的过渡位置
- 法辅（法师辅助）：和 ADC 保持半屏幕距离
