# 开团 / 反打 / 撤退判断

## 该开团的信号
- [ ] 你方人数优势（5v4 或更好）
- [ ] 对方关键技能在 CD（石头人 R、日女 R、卡牌 R）
- [ ] 对方 ADC 残血/没闪现
- [ ] 你们有猴子/石头人/蔚这类强开
- [ ] 龙/大龙马上要刷了（开团赢 = 免费打龙）

## 不该开团的信号
- 对面有风女/露露（反手太强）
- 对面有莫甘娜/塔姆（黑盾/吃了就白开）
- 自家 AD 还没到
- 没视野的地方

## 反打 vs 撤退
- 对方开了但没控到 → 立即反打
- 对方开了控到了前排 → 给前排治疗/盾，继续反打
- 对方开了秒了 AD → 立刻撤退（4v5 团战 90% 输）
- 对方有龙buff → 不要接团，守塔等buff过期
