# 目标优先级

## 团战目标优先级（从高到低）
1. **对方 ADC** — 持续输出源，活着 = 你们团不了
2. **对方中单** — 爆发输出源，一个 R 灭团
3. **对方主输出刺客** — 劫 / 泰隆 / 螳螂（如果他在切你们后排）
4. **对方辅助** — 但如果他是坦克辅助（泰坦/日女）→ 优先级降到最低
5. **对方上单** — 坦克型最后处理，战士型排名提前

## 不同阶段的目标切换
- **团战开打（前 3 秒）**：集火对方第一个被控到的人
- **团战中段（3-5 秒）**：此时 CD 全交，转火对方残血 carry
- **团战收割（5 秒+）**：追残血，但别追过河道（会被蹲）

## 永远不要追的目标
- 泰坦、日女、奥恩——追 20 秒也杀不死，还脱离团战
- 有中亚/复活甲（还活着）的人
- 往塔下跑的人
