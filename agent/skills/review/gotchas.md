# Review 坑点清单

## 数据解读
- **KDA 好看不等于打得好**：KDA 型打法（保命不输出）恰恰是低分段上不去分的第一大原因
- CS 差但参团率高不一定是牺牲——可能是无效游走，逛了一圈什么都没做
- 伤害最高不一定是 C 了——可能是无效伤害（打了被奶回来的坦克）
- 死亡多不一定是坑——如果是保 ADC 或者被集火换出 3-4 个技能

## 胜败分析
- **只有 20% 的游戏是"碾压局"**（一方完全克制），80% 是中间有翻盘机会的
- 如果输了但 KDA > 3 → 这不是你的锅
- 如果赢了但 KDA < 1 → 这是队友 carry 了你
- 复盘的重点是找到那 3-5 个"如果当时..."的转折点

## 建议输出
- **千万不要暗示"队友玩得差"**——复盘是给玩家的建议，不是来甩锅的
- 如果输了，改进点 > 表扬点（输了的复盘以改进为主）
- 如果赢了，表扬点 > 改进点（赢了的复盘以巩固为主）
- 下把重点练习的东西必须**只有一个**——给多个等于没给
