# 改进建议

## 改进框架：PDCA
- **Plan（规划）**：下把重点练什么（只设 1 个目标）
- **Do（执行）**：对局中有意识地练习这个目标
- **Check（检查）**：复盘时看这个目标达成没有
- **Adjust（调整）**：目标达成了就换下一个，没达成继续

## 常见改进方向速查

### 对线问题
→ 下把重点：前 10 分钟不追求击杀，追求完美补刀

### 意识问题
→ 下把重点：每波兵推完都看一次小地图

### 团战问题
→ 下把重点：团战开始后先锁定目标，再放技能

### 视野问题
→ 下把重点：每 3 分钟买 1 个真眼插在关键位置

### 心态问题
→ 下把重点：屏蔽所有聊天，只关注自己的操作

## 一周提升计划建议
- 如果 7 天都打同一件事，第 7 天你会变成本能
- 每天换一个改进目标 → 7 天后什么都没变
