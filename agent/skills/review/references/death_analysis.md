# 死亡原因分类和复盘

## 死亡分类
| 原因 | 典型场景 | 严重程度 |
|------|---------|---------|
| **被单杀** | 对线换血/越塔/拼输了 | 最严重——个人能力问题 |
| **被抓** | 对面打野/辅助来包夹 | 中——意识和视野问题 |
| **团战死** | 团队战中阵亡 | 轻——取决于死法 |
| **浪死** | 优势时浪/越塔/追残血 | 最严重——心态问题 |
| **保队友** | 替 carry 挡技能/掩护撤退 | 轻——团队贡献 |
| **被蹲** | 无视野时进草/推线过深 | 中——意识问题 |

## 复盘模板（按死亡原因）
```
第X次死亡 @ {time}分钟 — {原因}
- 具体情况：{简短描述}
- 根因：{意识/操作/信息缺失}
- 改进方案：{具体可执行的动作}
```

## 死亡复盘法则
- 如果前 3 波就死了 → 你玩的太激进/防 gank 意识差
- 如果全是团战死 → 站位有问题/集火目标不对
- 如果全是单杀 → 你不该玩这个英雄/这个对位
