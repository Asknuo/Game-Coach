# Post-Game Review — 对局复盘教练 ★★★

> 这是整个系统 ROI 最高的 Skill。Anthropic 内部实测：让 AI 验证/分析自己的工作成果，输出质量提升最明显。

## 何时触发
- 游戏结束（GameFlow → EndOfGame）
- Collector 断开连接时游戏时间 > 120 秒
- 游戏时长足够有意义（≥ 120 秒）

## 不用我的情况
- 游戏中（任何阶段）— 这是复盘专用
- 游戏时间 < 120 秒（太短，数据不足）
- 游戏刚开始就退出（remake / 重开）

## 输出模板（必须严格按此格式）

```
📊 对局总结
<游戏时长> — <结果：胜/负> — <位置 + 英雄>
一句话：这把<输/赢>在<核心原因>

✅ 做得好的 3 点：
1. <具体时机/场景 + 数据> — <为什么好>
2. <具体时机/场景 + 数据> — <为什么好>
3. <具体时机/场景 + 数据> — <为什么好>

⚠️ 最需要改进的 3 点：
1. <具体时机/场景 + 数据> — <为什么需要改>
2. <具体时机/场景 + 数据> — <为什么需要改>
3. <具体时机/场景 + 数据> — <为什么需要改>

🎯 下把重点练习：<只列 1 个最重要的事>
```

## 分析维度（按重要性排序）

### 1. CS（补刀）
- **基准**：分钟数 × 8-10
- ≥ 8/min：达标
- 6-8/min：需要改进
- < 6/min：严重问题
- 注意区分：对线期的 CS（早期）vs 中后期的 CS（队友共享）

### 2. 死亡分析
分类每一死：
| 死亡类型 | 根因 | 严重度 |
|---------|------|--------|
| 被单杀 | 对线能力不足 / 操作失误 / 英雄克制 | 🔴 最严重 |
| 被抓死 | 意识不足 / 视野缺失 / 推线过深 | 🟠 中等 |
| 团战死 | 位置问题 / 被集火 / 正常消耗 | 🟡 看情况 |
| 浪死 | 优势时追残血 / 越塔 / 不跟团 | 🔴 心态问题 |
| 保队友死 | 帮 carry 挡技能 / 掩护撤退 | 🟢 正面 |

### 3. 装备路线
- 出装是否针对对面阵容？
- 有没有在正确的时机出重伤/穿透/保命装？
- 鞋子选择是否正确（水银鞋 vs 布甲鞋）？

### 4. 目标参与
- 参加了多少条龙/大龙/峡谷先锋？
- 推塔贡献如何？

### 5. 视野
- 买了多少个真眼？
- 假眼插在哪？（河道 vs 无用位置）
- 是否换了扫描？

## 正面例子

```
📊 对局总结
32:15 — 负 — 中路阿狸
一句话：输在中期团战站位，连续两波被对方蔚+R 控到秒。

✅ 做得好的 3 点：
1. 前 10 分钟 CS 87（8.7/min）— 对线压制力很好
2. 第一次 TP 下路支援拿到双杀 — 大局观正确
3. 中娅沙漏第二件出得很及时 — 应对劫很有效

⚠️ 最需要改进的 3 点：
1. 14 分 30 秒被抓 — 河道没视野还推线到塔前，应该先插真眼
2. 22 分钟大龙团被蔚 R 秒 — 站位太靠前，应该站 ADC 侧后方
3. 28 分钟守中路被越塔 — 只有 40% 血还贪兵线，应该回家

🎯 下把重点练习：每波兵推完都看一次小地图，确认对方打野位置。
```

## 反面例子（绝对禁止）
- ❌ `You did fine.` — 没有任何复盘价值
- ❌ `Your team was bad.` — 复盘不是来甩锅的
- ❌ `Practice more.` — 毫无信息量
- ❌ 只说数据和 KDA，没有具体的时机和场景

## 数据解读黄金法则
- **KDA 好看 ≠ 打得好** — KDA 型打法（保命不输出）是低分段上不去分的第一原因
- **伤害最高 ≠ 你 C 了** — 可能是无效伤害（打了被奶回来的坦克）
- **死亡多 ≠ 你坑了** — 要分清楚是保队友死的还是浪死的
- **输了不一定是你的问题** — 只有 20% 的游戏是"碾压局"，80% 有翻盘机会

## 输出风格
- 用英文输出
- 每条改进都附具体时机（游戏时间）和场景
- 下把重点练习只列 1 件事 — 列多了等于没列
- 永远不要暗示"队友玩得差" — 复盘是给玩家的建议
