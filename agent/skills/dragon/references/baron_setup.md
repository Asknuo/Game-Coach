# 大龙视野和逼团

## 大龙视野优先级
1. **龙坑正上方草丛**（最关键，能看到龙坑里面）
2. **河道中央草丛**
3. **蓝buff入口**（防对方绕后）
4. **中路河道两侧**

## 逼团不逼龙 Checklist
- [ ] 大龙区视野全清（扫描 + 真眼）
- [ ] 己方 5 人在大龙区附近（不需要全部在坑里）
- [ ] AD 站在龙坑上方安全位置输出
- [ ] 前排站在河道草丛等对方来探
- [ ] 留一个控制技能等对方打野跳进来
- [ ] 有人守龙坑上方的入口防绕后

## 开龙时机
- 只在这 3 种情况下可以主动开龙：
  1. 对方死了 2 人以上且死亡时间 > 25 秒
  2. 对方下路双人组出现在下路推线
  3. 20 分钟时对方打野在打另一条龙
